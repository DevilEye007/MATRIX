## Features

### Core Animation
- **Authentic Matrix Effect**: Realistic falling characters with varying speeds and intensities
- **Smooth Performance**: Optimized 60 FPS animation using `requestAnimationFrame`
- **Dynamic Characters**: Mix of Katakana, Latin characters, and symbols

### Visual Enhancements
- **Cinematic Glow Effects**: Subtle text shadows and glow on leading characters
- **Dynamic Colors**: Customizable color schemes (Green, Blue, Red, Purple themes)
- **Retina Ready**: Crisp visuals on high-DPI displays
- **Motion Smoothing**: Velocity-based character movement for natural flow
